# 🧬 Alzheimer's Disease Risk Prediction from Genomic Data

> An end-to-end deep learning pipeline for predicting Alzheimer's disease risk 
> using DNA sequencing and genomic data.

---

## 📌 Problem Statement
Alzheimer's disease (AD) is a progressive neurodegenerative disorder with 
strong genetic components. This project builds a deep learning system to 
predict AD risk from genomic/DNA sequencing data, progressing from baseline 
ML models to advanced transformer-based architectures.

## 🎯 Project Goals
- Preprocess raw genomic data (VCF/SNP/FASTA formats)
- Train and evaluate multiple ML/DL models
- Achieve research-level performance with explainability (SHAP)
- Provide a deployable prediction API
